## Phase 2: The Data Mapping (file format spec)

Every module follows this schema:

- **Section Header** — e.g. `MODULE_03_RUG_PULLS`
- **Behavioral Label** — e.g. `INTERMITTENT_REINFORCEMENT`
- **Core Objective** — what the behavior accomplished for Derek
- **The Narrative** — the a→b→c of the pattern
- **Detailed Instance Analysis** — each instance with:
  - *Source*: `whatsapp_log` / `screenshot` / `user_testimony` / `report`
  - *Evidence ID*: `WA-####` (+ timestamp) or `SC-##`
  - *Direct Quote*: verbatim
  - *Analysis*: the mechanism
- **Psychological Impact** — effect on the victim
- **YAML Data Output** — machine-readable block at the end of each module
